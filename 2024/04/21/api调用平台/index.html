项目简介

用户可以注册登录并开通接口调用权限、浏览接口、在线调试,还能使用客户端SDK轻松在代码中调用接口。管理员可以接入并发布接口、统计分析各接口调用情况

api-f-wulaizhi 为前端项目。
api-client-sdk 为客户端调用sdk。
api-b-interface 为实际接口服务。
api-gateway 为网关实现用户调用的日志打印。签名认证，调用次数统计，黑白名单过滤。
api-b-wulaizhi 为后端主项目实现登录，注册，接口上线/退出，接口数据统计。

前端

Ant Design Pro
React
Ant Design Procomponents
Umi
Umi Requset (Axios 的封装)

后端

java Spring boot
Spring Boot Starter (SDK 开发)
Dubbo SpringCloud Gateway（网关、限流、日志实现）Nacos注册中心
